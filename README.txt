Adds a customisable form widget connected to Asoriba, allowing visitors to make payments to different donations.

== Description ==

The Asoriba Payments plugin is designed purposely to assist Churches and Christian Bodies receive donations and raise funds from their websites.You should have an account on Asoriba to make use of this plugin.
[Asoriba](https://asoriba.com) is church management app.

= Asoriba Features =
* Donations.
* Payments


== Installation ==

1. In your WordPress admin panel, go to Plugins > New Plugin, search for Asoriba and click 'Install now'. Alternatively, download the plugin and upload the contents of asoriba-payments.zip to your plugins directory, which is usually located at '/wp-content/plugins/'.
2. Activate the plugin.
3. You would find the Asoriba main menu on your admin menu. Click Asoriba - > Settings to authenticate your account. ANd Asoriba -> Dashboard to select your branch.
4. You'll find the Asoriba widget under the 'Appearance -> Widgets' menu. Drag it to a sidebar to activate.
5. Enter the title that would appear on the widget.
